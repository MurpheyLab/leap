# Project website template

The ``main`` branch is to host the code and document release. The website will be hosted under the ``gh-pages`` branch. 

To view the website locally, please clone the repo, switch to the ``gh-pages`` branch, and open the ``index.html`` file in the web browser. 

Note that, you can access the jupyter notebook tutorial on Google Colab with the following link: ``https://colab.research.google.com/github/MurpheyLab/[repo name]/blob/main/[tutorial notebook path]``.
